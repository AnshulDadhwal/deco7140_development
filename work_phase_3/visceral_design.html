Visceral Design

Explore the visual foundations of our design system, including page layouts, typography, and color schemes that create immediate emotional impact and visual appeal.

Page Layouts

Below are the wireframe layouts for key pages in our proposed design. Each layout has been carefully crafted to meet user needs and accessibility requirements identified in our research.

Color Scheme

Our color palette has been selected to ensure WCAG AA compliance while reflecting the client's brand values and creating emotional resonance with target users.

Primary Dark

#2C3E50

Used for headers and important text. Provides 12.63:1 contrast ratio on white backgrounds, exceeding WCAG AAA standards.

Primary Blue

#3498DB

Brand color used for interactive elements and call-to-action buttons. Tested for color-blind accessibility using simulation tools.

Light Grey

#ECF0F1

Background color for sections, providing subtle visual separation while maintaining readability.

Accent Red

#E74C3C

Used sparingly for alerts and important notices. Paired with icons for users with color vision deficiencies.

Typography

Our typography system ensures readability across all devices and supports our design hierarchy while meeting accessibility requirements.

Heading 1

Font: Roboto, sans-serif
Size: 3.5rem (56px) desktop / 2.25rem (36px) mobile
Weight: 700 (Bold)
Usage: Page titles only. Large size ensures visibility and establishes clear information hierarchy.

Heading 2

Font: Roboto, sans-serif
Size: 2.5rem (40px) desktop / 1.75rem (28px) mobile
Weight: 700 (Bold)
Usage: Section headings. Provides clear content structure for screen reader users.

Heading 3

Font: Roboto, sans-serif
Size: 1.875rem (30px) desktop / 1.375rem (22px) mobile
Weight: 500 (Medium)
Usage: Subsection headings and card titles.

Body Text - This is the standard paragraph text used throughout the site. Line height is set to 1.6 for optimal readability, and font size is minimum 16px to meet accessibility guidelines.

Font: Roboto, sans-serif
Size: 1.125rem (18px) desktop / 1rem (16px) mobile
Weight: 400 (Regular)
Line Height: 1.6
Usage: All body content. Never smaller than 16px to ensure readability for users with visual impairments.

Small Text - Used for captions, metadata, and supplementary information.

Font: Roboto, sans-serif
Size: 0.875rem (14px)
Weight: 400 (Regular)
Usage: Image captions and minor details. Maintains minimum readable size.
